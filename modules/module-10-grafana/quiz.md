# Quiz - Review questions and answers

## Questions

1. What is the role of a Grafana data source?
2. When is Explore useful?
3. Why should dashboards have a clear audience?
4. What makes a visualization trustworthy?
5. Why can too many panels harm incident response?
6. How should visualization type be selected?
7. Why does dashboard performance matter?

## Answers

1. A data source connects Grafana to a backend that stores or exposes telemetry.
2. Explore is useful for interactive investigation before creating or refining dashboard panels.
3. Audience determines which questions, detail level and layout are useful.
4. Clear units, labels, filters, thresholds and correct queries make visualizations trustworthy.
5. Too many panels increase cognitive load and slow down first response.
6. Select the visualization type based on the question: trends, current values, distributions or records.
7. Slow dashboards are less likely to be trusted or used during incidents.
