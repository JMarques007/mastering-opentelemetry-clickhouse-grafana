# Exercise - Checkout dashboard plan

## Goal

Plan a Grafana dashboard that helps operators understand checkout health.

## Instructions

Define three panels:

1. Request rate.
2. p95 latency.
3. Error rate.

For each panel, specify:

- data source;
- query intent;
- unit;
- visualization type;
- threshold or comparison;
- action an operator should take if it looks abnormal.

## Expected outcome

A strong answer connects each panel to an operational decision. It uses appropriate visualizations, clear units and a realistic investigation path into logs or traces.

## Instructor notes

If learners add many panels, ask which three would matter during the first two minutes of an incident. Use this to reinforce dashboard purpose and audience.
