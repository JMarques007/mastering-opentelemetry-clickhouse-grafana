# Official references

- [OpenTelemetry Documentation](https://opentelemetry.io/docs/) - Main documentation for telemetry concepts and implementation.
- [Grafana Alerting](https://grafana.com/docs/grafana/latest/alerting/) - Alerting documentation relevant to incident detection and response.
- [ClickHouse Documentation](https://clickhouse.com/docs) - Storage and query documentation for telemetry investigation.
- [W3C Trace Context](https://www.w3.org/TR/trace-context/) - Trace identity propagation standard.

Use these references when connecting the full incident workflow back to official observability building blocks.
