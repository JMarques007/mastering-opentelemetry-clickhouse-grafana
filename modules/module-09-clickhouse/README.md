# Module 09 - ClickHouse

This module introduces ClickHouse as a columnar analytical database for high-volume telemetry storage and investigation.

## Learning objectives

- Explain why columnar storage fits observability workloads.
- Understand ingestion, partitioning, ordering and retention.
- Query telemetry data with SQL.
- Recognize operational considerations for ClickHouse-backed observability.
