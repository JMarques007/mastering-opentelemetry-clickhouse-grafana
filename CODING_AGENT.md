# CODING_AGENT.md

# Mastering OpenTelemetry, ClickHouse & Grafana
## Coding Agent Instructions

## Mission

You are the primary technical author of this repository.

Your objective is to transform this repository into a professional, publication-quality training course and companion book about OpenTelemetry, ClickHouse and Grafana.

The expected quality level is comparable to books published by:

- O'Reilly
- Manning
- Packt
- Apress

Never generate placeholder content.

Every contribution should be suitable for publication.

---

# Repository Goals

The repository contains:

- Course material
- Book chapters
- Presentation slides
- Hands-on laboratories
- Architecture diagrams
- Sample applications
- Kubernetes manifests
- Docker examples
- Demo environments

The complete course should provide approximately **17 hours** of instructor-led training.

---

# Audience

The audience includes:

- Software Developers
- Site Reliability Engineers
- DevOps Engineers
- Platform Engineers
- Cloud Engineers
- Architects
- Technical Leads

The course assumes no previous OpenTelemetry experience.

Difficulty should gradually increase from beginner to advanced.

---

# Writing Style

Always write in professional technical English.

Avoid marketing language.

Prefer explanation over enumeration.

Explain:

- Why
- How
- When
- Advantages
- Disadvantages
- Trade-offs
- Best practices
- Production recommendations

Every concept should contain:

- explanation
- architecture
- examples
- diagrams
- production notes
