# Official references

- [Grafana Alerting](https://grafana.com/docs/grafana/latest/alerting/) - Grafana alerting concepts and configuration.
- [Prometheus Alerting](https://prometheus.io/docs/alerting/latest/overview/) - Prometheus alerting overview.
- [Prometheus Alerting Rules](https://prometheus.io/docs/prometheus/latest/configuration/alerting_rules/) - Rule configuration documentation.
- [OpenTelemetry Metrics](https://opentelemetry.io/docs/specs/otel/metrics/) - Metric model used by many alerting workflows.

Use these references when teaching rule design, alert routing and metric-based symptoms.
