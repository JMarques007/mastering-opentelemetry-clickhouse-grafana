# Course Roadmap

This roadmap tracks the planned build-out of the training repository for Mastering OpenTelemetry, ClickHouse and Grafana.

## Phase 1 - Repository Foundation

- Establish repository structure and contribution conventions.
- Define the module authoring template.
- Prepare documentation and GitHub Pages structure.
- Capture the course scope, audience and working principles.

## Phase 2 - Core Observability Concepts

- Module 01: Introduction to Observability.
- Module 02: OpenTelemetry Architecture.
- Shared diagrams for signals, pipelines and telemetry flow.
- Foundational labs that use a small demo service.

## Phase 3 - OpenTelemetry Practice

- Module 03: OpenTelemetry Collector.
- Module 04: Logs.
- Module 05: Metrics.
- Module 06: Traces.
- Module 07: Context Propagation.
- Module 08: Instrumentation.
- Reusable Collector configurations and lab exercises.

## Phase 4 - Storage and Visualization

- Module 09: ClickHouse.
- Module 10: Grafana.
- Module 11: Dashboards.
- Module 12: Alerting.
- Production-ready examples for schema design, dashboards and alert rules.

## Phase 5 - Production Readiness

- Module 13: Best Practices.
- Module 14: Production Case Study.
- Operational guidance for reliability, cost, governance and troubleshooting.
- Final workshop path and instructor checklist.

## Maintenance Principles

- Keep Markdown as the source of truth.
- Prefer editable diagrams over exported-only images.
- Keep labs reproducible from a clean checkout.
- Review module status whenever course content changes.
