# Official references

- [OpenTelemetry Collector Documentation](https://opentelemetry.io/docs/collector/) - Main documentation for Collector concepts and usage.
- [Collector Configuration](https://opentelemetry.io/docs/collector/configuration/) - Official configuration model for receivers, processors, exporters and pipelines.
- [OpenTelemetry Collector Contrib](https://github.com/open-telemetry/opentelemetry-collector-contrib) - Community Collector components used in many production deployments.
- [OpenTelemetry Specification](https://opentelemetry.io/docs/specs/otel/) - Normative behavior for OpenTelemetry signals and SDKs.

Use these references when building labs, explaining component responsibilities or validating Collector configuration examples.
