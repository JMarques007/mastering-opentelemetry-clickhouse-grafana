# Official references

- [OpenTelemetry Traces](https://opentelemetry.io/docs/concepts/signals/traces/) - Conceptual documentation for traces and spans.
- [OpenTelemetry Trace Specification](https://opentelemetry.io/docs/specs/otel/trace/) - Normative trace API and SDK behavior.
- [W3C Trace Context](https://www.w3.org/TR/trace-context/) - Standard propagation format for distributed trace identity.
- [Grafana Tempo Documentation](https://grafana.com/docs/tempo/latest/) - Official documentation for Grafana Tempo trace storage and querying.

Use these references to teach span structure, trace propagation, sampling and trace exploration.
