# Module 06 - Traces

This module explains distributed traces as request journeys and teaches how spans, attributes and timing reveal production behavior.

## Learning objectives

- Explain traces, spans and parent-child relationships.
- Understand span attributes and status.
- Use traces to identify latency and failure points.
- Connect traces with logs and metrics.
