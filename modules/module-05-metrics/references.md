# Official references

- [OpenTelemetry Metrics](https://opentelemetry.io/docs/specs/otel/metrics/) - OpenTelemetry metric data model and instrument behavior.
- [OpenTelemetry Semantic Conventions](https://opentelemetry.io/docs/specs/semconv/) - Standard metric and attribute names.
- [Prometheus Data Model](https://prometheus.io/docs/concepts/data_model/) - Prometheus metric naming, labels and samples.
- [Grafana Documentation](https://grafana.com/docs/) - Grafana documentation for dashboards and metric visualization.

Use these references when discussing instruments, labels, cardinality and metric-based dashboards.
