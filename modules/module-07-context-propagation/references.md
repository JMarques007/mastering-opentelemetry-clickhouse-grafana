# Official references

- [W3C Trace Context](https://www.w3.org/TR/trace-context/) - Standard HTTP trace context propagation format.
- [OpenTelemetry Context Propagation](https://opentelemetry.io/docs/concepts/context-propagation/) - OpenTelemetry guidance for propagation concepts.
- [OpenTelemetry Baggage Specification](https://opentelemetry.io/docs/specs/otel/baggage/) - Specification for propagated key-value baggage.
- [OpenTelemetry Language SDKs](https://opentelemetry.io/docs/languages/) - Language-specific instrumentation and propagation guidance.

Use these references for standards-based propagation, baggage safety and language-specific examples.
