# Module 06 - Traces

## Duration

Approximately 1 hour.

## Purpose

This module teaches distributed traces as request-level evidence for causality, latency and dependency investigation.

## Main topics

- Spans, attributes and status.
- Trace timelines and critical path.
- Sampling trade-offs.
- Correlation with logs and metrics.
