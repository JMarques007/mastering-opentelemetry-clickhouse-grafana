# Official references

- [OpenTelemetry Documentation](https://opentelemetry.io/docs/) - Main OpenTelemetry documentation for instrumentation and operations.
- [OpenTelemetry Semantic Conventions](https://opentelemetry.io/docs/specs/semconv/) - Standard telemetry naming and attributes.
- [Grafana Dashboard Best Practices](https://grafana.com/docs/grafana/latest/dashboards/build-dashboards/best-practices/) - Official dashboard design guidance.
- [CNCF TAG Observability](https://tag-observability.cncf.io/) - CNCF observability community resources.

Use these references when defining platform standards, reviewing telemetry quality or teaching governance.
