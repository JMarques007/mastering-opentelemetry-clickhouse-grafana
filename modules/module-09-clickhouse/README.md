# Module 09 - ClickHouse

## Duration

Approximately 1 hour.

## Purpose

This module explains why ClickHouse fits analytical telemetry workloads and how schema, ingestion and retention decisions affect observability performance.

## Main topics

- Columnar storage for telemetry.
- Schema design and order keys.
- Batch ingestion and retention.
- SQL investigation patterns.
