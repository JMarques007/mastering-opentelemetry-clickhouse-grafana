# Official references

- [OpenTelemetry Logs](https://opentelemetry.io/docs/specs/otel/logs/) - OpenTelemetry log data model and behavior.
- [OpenTelemetry Semantic Conventions](https://opentelemetry.io/docs/specs/semconv/) - Standard names for common telemetry attributes.
- [Grafana Loki Documentation](https://grafana.com/docs/loki/latest/) - Official Loki documentation for log aggregation and querying.
- [OWASP Logging Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Logging_Cheat_Sheet.html) - Security-focused guidance for safe logging.

Use these references to teach log structure, safe fields, correlation and retention decisions.
